Ciberseguridad: El Malware

El malware es un programa dañino creado para alterar el funcionamiento de dispositivos, robar información o controlar sistemas sin permiso.

¿Cómo funciona? (EJEMPLOS)

Todos los tipos de malware siguen el mismo patrón básico: Su dispositivo se infecta después de descargar o instalar software malicioso involuntariamente, por lo general al hacer clic en un enlace infectado o al visitar un sitio web infectado:

  • Correos electrónicos maliciosos: Envían archivos adjuntos o enlaces falsos que instalan malware al abrirlos.
  • Descargas engañosas: Hacen que los usuarios descarguen programas aparentemente legítimos que contienen malware.
  • Redes Wi-Fi inseguras: Configuran puntos de acceso falsos para interceptar datos y propagar malware.

¿Quién crea el malware?

  • Hackers con malas intenciones.
  • Grupos criminales organizados.
  • Incluso gobiernos en casos de ciberguerra.

¿Para qué sirve el malware?

  • Robar datos personales o financieros.
  • Secuestrar sistemas para pedir rescates (*ransomware*).
  • Espiar actividades de usuarios o empresas.

Comparativa de ataques de malware: 2020 vs. 2023

Año Tipos de ataques más comunes Cantidad estimada de ataques
2020 Ransomware, spyware, phishing ~1.5 millones por mes
2023 Ransomware, ataques a IoT, phishing avanzado ~3 millones por mes

¿Cómo protegerse?

  1. Instalar software antivirus y mantenerlo actualizado.
  2. No abrir enlaces o archivos sospechosos.
  3. Actualizar regularmente el sistema operativo y las aplicaciones.
  4. Instalar un cortafuegos
